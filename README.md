# Azure AD User Lifecycle Management

## Overview
This project demonstrates hands-on user lifecycle management in Microsoft Entra ID (Azure AD) via the Azure portal. Tasks include:

- Creating new user accounts
- Modifying user properties
- Resetting passwords
- Revoking access
- Deleting user accounts

The project showcases practical **Identity and Access Management (IAM)** skills and **Role-Based Access Control (RBAC)** implementation.
